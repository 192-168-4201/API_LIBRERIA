# API Librería

Una API RESTful para gestionar una librería, construida con Node.js, Express, MongoDB y JWT para autenticación.

## Características

- Gestión de libros, autores y usuarios.
- Autenticación JWT.
- CRUD completo para libros y autores.
- Base de datos MongoDB.

## Tecnologías

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT (JSON Web Tokens)
- Bcryptjs para hash de contraseñas
